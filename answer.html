How do I return my order?

Sep 15, 2021 • Watches
If you are within our return policy , you will need 3 things to return your order:
  • The item(s) you are returning
  • The return shipping label
  • The return detail page

Fill out the return detail page and include it in your package with the item(s) you are returning. Paste the return shipping label on the package.

Details on how to receive a return shipping label can be found below.
  • Enter your order information to track your order here. Use your order number, order email address, and the billing ZIP code on the order.
  • In the Order Status window click on, "Return this item" for the items you will be returning.
  • Select the reason for return for each item you will be returning.
  • Download your return shipping label.
  • Print out the label and paste it on your package.

Bring your package to your local FedEx store or drop box and track the package to know when to expect your refund.
